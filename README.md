### Installation and Usage
1. Install the Pygame Zero library using PIP command:
    `pip install pgzero`
2. Import the project into your IDE and click run, or run with the following command:
    `pgzrun covidhero.py`
For more detailed instructions, please see [Pygame Zero Documentation](https://pygame-zero.readthedocs.io/en/stable/index.html).
